# Hello TypeScript

Simple hello world project for running TypeScript with Node.js

## Running the Project

1. Run `yarn install`
2. Run `yarn test`

## Running the Project

1. node App.ts
2. Run `yarn test`

## Debugging the Project

1. Open the project in VS Code
1. Open the `app.ts` file
1. Set a breakpoint on one of the lines of code
1. Press `F5`
